# 個人房貸試算器

## 專案簡介

這是一個使用 C# Windows Forms 開發的個人房貸試算器。  
使用者輸入房屋總價、自備款比例、年利率、貸款年限與寬限期，程式會自動計算每月應繳金額、首期利息、首期本金、總利息支出與總還款金額。

## 功能說明

- 輸入房屋總價（元）與自備款比例（%）自動計算貸款總金額
- 採用**本息均攤**公式計算每月應繳金額
- 支援**寬限期**設定（寬限期內只繳利息不還本金）
- 所有金額格式化顯示（含千分位逗號與小數點後兩位）
- 輸入驗證：非數字、負值、超出範圍皆顯示錯誤訊息

## 計算公式

### 貸款總金額
```
貸款金額 = 房屋總價 × (1 - 自備款比例%)
```

### 每月應繳金額（本息均攤）
```
月利率 r = 年利率 / 12
還款月數 n = (貸款年限 - 寬限期) × 12
每月應繳 = 貸款金額 × r × (1+r)^n / ((1+r)^n - 1)
```

### 總利息支出
```
總還款金額 = 每月應繳 × 還款月數 + 寬限期每月利息 × 寬限月數
總利息 = 總還款金額 - 貸款金額
```

## 輸入欄位說明

| 欄位 | 說明 | 預設值 |
|------|------|--------|
| 房屋總價 | 整數或浮點數，單位：元（NT$） | 10,000,000 |
| 自備款比例 | 0 ~ 99 的百分比 | 20 |
| 年利率 | 以百分比表示（例：2.15） | 2.15 |
| 貸款年限 | 以年為單位（例：20、30） | 30 |
| 寬限期 | 選填，前幾年只繳利息（預設 0） | 0 |

## 執行說明

1. 使用 Visual Studio 開啟專案（需 .NET Framework）
2. 按下 `F5` 或點選「啟動」執行程式
3. 依序輸入各欄位數值（欄位已有預設值可直接計算）
4. 點選「計算」查看試算結果
5. 點選「清除」重置為預設值

## 開發環境

- 語言：C#
- 框架：.NET Framework（Windows Forms）
- IDE：Visual Studio 2022
